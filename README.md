# Sales Data Analysis Dashboard

## Overview

This project analyzes sales data using Python and provides business insights through data cleaning, exploratory data analysis, visualizations, and an interactive Streamlit dashboard.

## Features

* Data Cleaning and Preprocessing
* Sales Trend Analysis
* Region-wise Sales Analysis
* Product Category Analysis
* Interactive Dashboard using Streamlit
* Data Visualization using Matplotlib

## Technologies Used

* Python
* Pandas
* Matplotlib
* Streamlit

## Project Structure

Sales-Data-Analysis/
├── app.py
├── sales_analysis.py
├── data/
├── visualizations/
└── README.md

## How to Run

Install dependencies:

pip install pandas matplotlib streamlit

Run analysis:

python sales_analysis.py

Run dashboard:

streamlit run app.py

## Outcomes

* Identified sales trends
* Compared regional performance
* Analyzed product category sales
* Built an interactive analytics dashboard
